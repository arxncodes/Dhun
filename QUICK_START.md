### Accessing Admin Dashboard
1. Click **"Admin Dashboard"** in the sidebar (only visible to admins)
2. View statistics and manage content

### Uploading Music/Podcasts
1. Click **"Upload Track"** button
2. Select content type (Music or Podcast)
3. Choose an audio file from your computer
4. Fill in the details:
   - **Title**: Track name (required)
   - **Artist** (for music) or **Podcast Name** (for podcasts)
   - **Category**: Genre or topic (optional)
   - **Cover Image URL**: Link to cover art (optional)
5. Click **"Upload"**

### Managing Users
1. Go to the **"Users"** tab
2. Use the dropdown to change user roles between "user" and "admin"

### Deleting Tracks
1. Go to the **"Tracks"** tab
2. Click the trash icon next to any track to delete it

## ğŸ› Troubleshooting

### Can't hear audio?
- Check your device volume
- Make sure the player volume isn't muted
- Try a different browser (Chrome recommended)

### Upload not working?
- Ensure you're logged in as an admin
- Check file size (keep under 10MB for best performance)
- Use common audio formats (MP3, WAV, OGG)

### Can't see admin features?
- Make sure you're the first registered user
- Or ask an existing admin to promote your account
