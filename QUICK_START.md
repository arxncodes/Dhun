3. Enter artist name (required)
4. Select category (required)
5. Upload one cover image (optional)

## ğŸ’¡ Pro Tips

1. **Filename Strategy**: Name files descriptively before uploading
   - Good: `01_hey_jude.mp3`, `02_let_it_be.mp3`
   - Bad: `track1.mp3`, `audio.mp3`

2. **Cover Images**: Prepare cover art in advance
   - Format: JPG or PNG
   - Size: 500x500 or 1000x1000 pixels
   - Square aspect ratio (1:1)

3. **Batch Processing**: Group similar tracks together
   - Same artist â†’ Bulk upload
   - Same album â†’ Bulk upload
   - Different artists â†’ Single upload

4. **Quality Control**: Review before uploading
   - Check track titles
   - Verify artist name spelling
   - Confirm category selection
   - Preview cover image
