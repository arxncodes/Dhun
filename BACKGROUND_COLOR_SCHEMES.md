### Animation Characteristics
- **Duration**: 20 seconds per cycle
- **Easing**: ease-in-out (smooth acceleration/deceleration)
- **Loop**: Infinite
- **Direction**: Diagonal (135deg)
- **Size**: 400% x 400% (large for smooth transitions)

## Browser Rendering

### CSS Properties Used
```css
/* Efficient properties for GPU acceleration */
background: linear-gradient(...)  /* GPU accelerated */
background-size: 400% 400%        /* No layout impact */
background-position: ...          /* GPU accelerated */
animation: ...                    /* GPU accelerated */
position: fixed                   /* No reflow */
z-index: -1                       /* Behind content */
```

### Why It's Fast
1. **Fixed positioning**: No layout recalculation
2. **GPU acceleration**: Uses compositor thread
3. **No JavaScript**: Pure CSS, no overhead
4. **Single element**: Minimal DOM impact
5. **Background-only**: Doesn't affect text rendering

## Design Philosophy

### Light Theme
**Inspiration**: Clear sky, calm ocean, fresh air  
**Goal**: Energize and uplift users during daytime listening  
**Approach**: Bright, airy colors with smooth transitions

### Dark Theme
**Inspiration**: Night sky, deep space, quiet evening  
**Goal**: Provide comfortable, immersive experience for nighttime  
**Approach**: Subtle variations in dark tones for depth without distraction
